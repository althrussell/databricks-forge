# UX + shadcn Rules

- Use shadcn primitives: Card, Table, Tabs, Dialog, Sheet, Badge, Button, Skeleton, Toast.
- Page layout:
  - header with title + scope controls
  - KPI cards row
  - main content with a table or detail panel
- Always show “Why ranked” inline for each candidate.
- Provide copy-to-clipboard for SQL and IDs.
- Make the primary CTA visually dominant.
